## Get the Repo
How to pull this repo onto your machine:

AT NICAR:
1. Open R Studio
2. Make sure you aren't already in a project (check in the top right, if you are, close the project)
3. Open the terminal tab
4. Make sure you are in the home folder, type `pwd`. If you aren't you can get there by backing out using `cd ..` until you get there
5. Navigate to the folder we want: `cd Desktop/hands_on_classes/20260306-r-2-analysis-and-plotting-repeat` OR `cd Desktop/hands_on_classes/20260306-r-3-gathering-and-cleaning`
6. clone our git repo into this folder: `git clone COPY FROM GITHUB HERE`

On your own machine:
Follow the same steps but navigate to whatever folder you want to store this repo in instead of the hands on folder specifically.
